# 韶光-找回生活点滴[去模糊图像系统]

### “韶光”以抓住**生活的点滴**为创意初心，基于飞桨的深度学习框架，利用生成对抗网络处理运动图像模糊的问题，与移动端相结合，致力于解锁大众拍摄的“新姿势”，推动AI走进生活的每一个角落。
